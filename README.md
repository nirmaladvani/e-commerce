```bash

cd e-commerce
npm install
npm start
```

## 🧪 Tech Stack

- **React** (Functional Components + Hooks)
- **CSS** (Custom styling per component)
- **Node.js** (Backend server with Express)
- **MongoDB** (NoSQL database for storing user and product data)
